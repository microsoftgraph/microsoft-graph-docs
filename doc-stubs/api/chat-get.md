# Get chat
Namespace: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

## HTTP request

}

``` http

```

|:---|:---|

If successful, this method returns a `200 OK` response code and a [chat](../resources/chat.md) object in the response body.
